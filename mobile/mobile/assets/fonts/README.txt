Please download and add the following font files to this directory for the app to display correctly:

1. Poppins font family:
   - Poppins-Regular.ttf
   - Poppins-Medium.ttf
   - Poppins-SemiBold.ttf
   - Poppins-Bold.ttf

2. Roboto font family:
   - Roboto-Regular.ttf
   - Roboto-Medium.ttf
   - Roboto-Bold.ttf

These fonts can be downloaded from Google Fonts:
- Poppins: https://fonts.google.com/specimen/Poppins
- Roboto: https://fonts.google.com/specimen/Roboto

Alternatively, you can use the google_fonts package in the app instead of physical font files.
